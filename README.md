# SCC-240 - Bases de Dados

Repositório destinado aos projetos, exercícios e materiais desenvolvidos durante a disciplina **SCC-240 – Bases de Dados**, ministrada no Instituto de Ciências Matemáticas e de Computação (ICMC-USP).

## Sobre a disciplina

A disciplina tem como objetivo apresentar os conceitos fundamentais de bancos de dados, abordando desde a modelagem conceitual até a implementação de sistemas utilizando Sistemas Gerenciadores de Banco de Dados (SGBDs). O foco é desenvolver a capacidade de projetar e construir sistemas de informação baseados em boas práticas de modelagem e organização dos dados.

## Conteúdo estudado

Ao longo da disciplina são abordados temas como:

- Conceitos fundamentais de Banco de Dados;
- Estrutura e funcionamento de um SGBD;
- Modelagem Conceitual utilizando o Modelo Entidade-Relacionamento (MER);
- Modelo Relacional;
- Mapeamento MER → Modelo Relacional;
- Álgebra Relacional;
- Normalização;
- Linguagem SQL;
- Introdução a Transações;
- Desenvolvimento de Sistemas de Banco de Dados. 

## Tecnologias utilizadas

- SQL
- Sistemas Gerenciadores de Banco de Dados (SGBDs)
- Modelagem Entidade-Relacionamento (MER)
- Modelo Relacional
- Álgebra Relacional
